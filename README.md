# engwithpeymar

This is an AI-powered English mentor bot for Telegram. It helps users practice English through conversations, daily vocabularies , and personalized corrections.

## Features

**Word of the Day**: generates a new word with definition and context suited to your level.
**Assessment**: runs quick quizzes to test your knowledge.
**Roleplay**: simulates real-world scenarios (like a job interview or ordering coffee) for practice.
**Corrections**: you can send any sentence, and the bot will correct the grammar and explain the mistakes.
**Vocabulary List**: saves words you want to remember so you can review them later.
